# FlexChess - 西洋棋兵陣學

透過西洋棋陣型來學習 CSS Flexbox 佈局的互動式遊戲。

## 🚀 專案簡介
這是一個結合西洋棋策略與 CSS Flexbox 技術的學習工具，幫助初學者以直觀的方式掌握複雜的佈局屬性。

（本專案作為「六角學院 2026 30 天軟體工程師體驗營」之作業）

## ✨ 主要功能
*   **互動式挑戰**：透過即時編輯 CSS 來完成棋陣佈局。
*   **關卡系統**：從基礎到進階，循序漸進學習 `justify-content`, `align-items`, `flex-wrap` 等屬性。
*   **視覺化回饋**：即時預覽佈局變化。
*   **現代 UI 設計**：響應式介面，支援行動裝置與桌面版。

## 🛠 技術棧
本專案使用以下技術進行開發：
*   **Vue.js 3** (Composition API)
*   **Tailwind CSS**
*   **Three.js** (用於 3D 視覺效果)
*   **原生 JavaScript / CSS3**

## 📦 如何執行
直接在瀏覽器開啟 `index.html` 即可開始遊玩，無需複雜的建置步驟。
